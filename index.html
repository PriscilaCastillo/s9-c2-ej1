6. Como iterar los arreglos , de manera imperativa(for) y de manera declarativa (map y forEach)

const numeros = [1, 2, 3];

Imperativo
for (let i = 0; i < numeros.length; i++) {
console.log(numeros[i]);
}

Declarativa

forEach:
numeros.forEach(function(numero) {
console.log(numero);
});

Map:
const duplicados = numeros.map(function(numero) {
return numero * 2;
});
console.log(duplicados);

7. Diferencias con ejemplos de map y forEach

Map devuelve un nuevo arreglo con los resultados de la iteración, forEach ejecuta la función en cada elemento del arreglo sin crear un nuevo arreglo.
